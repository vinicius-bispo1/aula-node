# 🚀 API de Autenticação e Usuários

API desenvolvida em Node.js para gerenciamento de usuários com autenticação via JWT e criptografia de senhas com bcrypt.

---

## 📌 Funcionalidades

- Cadastro de usuários
- Login com geração de token JWT
- Listagem de usuários (protegida)
- Atualização de usuário
- Remoção de usuário

---

## 🧱 Tecnologias utilizadas

- Node.js
- Express
- PostgreSQL
- bcrypt
- jsonwebtoken
- dotenv

---

## ⚙️ Pré-requisitos

- Node.js instalado
- PostgreSQL instalado
- Gerenciador de pacotes (npm ou yarn)
